Good CSS

Hello! Welcome to the demo page for Good CSS.

This demo page uses Good CSS as the base with a few additional styles applied on top of it (mainly for formatting). Good CSS is meant to provide a starting place when first building out a web application, and additional customization is highly encouraged.

Toggle light and dark mode in Google Chrome by typing this command into the browser bar: chrome://settings/appearance.
